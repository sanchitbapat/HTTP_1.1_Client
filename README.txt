An HTTP 1.1 Client that sends the request and receives a response. The client handles automatic redirection for a maximum of 5 redirects. It handles chunked encoding and displays the decoded data.

Steps:
1. Make sure JAVA is installed.
2. Run the Makefile via command prompt.
3. On the command prompt type the following:
./client <method> <url>

<method> is any of the standard http methods (GET, POST, PUT, etc.)
<url> is the website you want ot access.

4. The source of the webpage is diplayed.
